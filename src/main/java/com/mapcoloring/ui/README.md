# ui 模块 — JavaFX 图形界面

## 模块职责

提供地图着色的图形化展示和交互。**此模块可以使用 JavaFX 类库**（课程允许图形界面调库）。

## 类列表

| 类 | 父类 | 职责 |
|----|------|------|
| `MainApp` | `Application` | 程序入口，创建主窗口 |
| `FrontPanel` | `BorderPane` | 着色界面：左侧地图 + 右侧控制面板 |
| `MapCanvas` | `Pane` | 地图画布：渲染省份多边形、交互控制 |

## MainApp — 主窗口

- 创建 1000×620 的窗口，标题"中国地图着色"
- 直接展示 `FrontPanel`（无标签页，简洁设计）

## FrontPanel — 着色界面

**布局**：`BorderPane`，左侧地图画布 + 右侧控制面板（200px 宽）

**控制面板按钮**：
- 三个算法按钮：贪心着色、贪心+排序、回溯求最优
- 统计信息：颜色总数、算法耗时、当前算法名
- 重新加载按钮

**启动行为**：构造时自动加载 `data/china.map`

## MapCanvas — 地图画布

**架构设计**：继承 `Pane`，内部包含 `Canvas`。重写 `layoutChildren()` 实现自适应窗口缩放——每次窗口大小变化时自动重新计算缩放比例和偏移量，触发重绘。

### 渲染流程

1. **坐标变换**：将省份的多边形顶点从数据坐标转换为画布像素坐标
2. **多边形填充**：根据 `graph.getColor(i)` 从 8 色调色板取色，未着色显示浅灰
3. **边框描边**：深灰色边框，鼠标悬停时变红色加粗
4. **名称标注**：在多边形中心位置绘制省份名称（微软雅黑字体）

### 交互功能

| 操作 | 实现 |
|------|------|
| 滚轮缩放 | 以鼠标位置为中心，缩放因子 1.1/0.9 |
| 拖拽平移 | 记录鼠标按下位置，计算偏移量 |
| 悬停高亮 | 射线法点-多边形碰撞检测，遍历所有省份 |

### 点-多边形检测（射线法）

从鼠标位置向右发射水平射线，统计与多边形各边的交点数量。奇数次交点在多边形内部，偶数次在外部。时间复杂度 O(n)，n 为多边形边数。

### 自适应缩放

`layoutChildren()` 在以下时机被 JavaFX 自动调用：
- 窗口首次显示
- 窗口大小改变
- 布局约束变化

每次调用时：获取当前可用空间 → 计算数据边界框 → 等比缩放适配画布 → 重绘全部省份。
